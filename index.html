/ FAQ

คำถามที่พบบ่อย

ข้อมูลบน Cloud Storage และ Server ปลอดภัยแค่ไหน?+

ระบบทั้งหมดตั้งอยู่บน Data Center มาตรฐาน CS Loxinfo รับประกัน Uptime 99.9% ป้องกันข้อมูลสูญหายด้วย Hardware RAID และมีระบบ Backup อัตโนมัติทุกวัน พร้อมทีมงานดูแลระบบตลอด 24 ชั่วโมง

เครื่องคำนวณ License คำนวณแม่นยำแค่ไหน?+

คำนวณตามกฎ Core-Based Licensing ที่ Microsoft เผยแพร่เป็นสาธารณะ (ขั้นต่ำ 8 core ต่อ processor, ขั้นต่ำ 16 core ต่อเครื่อง) ใช้สำหรับวางแผนงบประมาณเบื้องต้น ราคาจริงและเงื่อนไข Software Assurance อาจแตกต่างกันตามผู้จำหน่าย แนะนำให้ติดต่อทีมงานเพื่อขอใบเสนอราคาที่แม่นยำ

ให้บริการนอกพื้นที่กรุงเทพฯ ด้วยหรือไม่?+

ให้บริการทั้งในเขตกรุงเทพฯ และต่างจังหวัด งานติดตั้งหน้างานและ On-site Support ครอบคลุมทั่วประเทศ ขึ้นอยู่กับขอบเขตของแต่ละโปรเจกต์

มีค่าใช้จ่ายในการให้คำปรึกษาเบื้องต้นหรือไม่?+

ไม่มีค่าใช้จ่าย ทีมวิศวกรจะเข้าไปประเมินหน้างานและเสนอแนวทางเบื้องต้นให้ก่อน จึงจะสรุปขอบเขตงานและใบเสนอราคาอย่างเป็นทางการ

Managed Services กับ MA (Maintenance Agreement) ต่างกันอย่างไร?+

MA คือสัญญาดูแลบำรุงรักษาอุปกรณ์ตามรอบเวลา ส่วน Managed Services ครอบคลุมกว้างกว่า คือการดูแลระบบเชิงรุกต่อเนื่อง ตรวจสุขภาพระบบ และให้คำปรึกษาระหว่างการใช้งาน

หากระบบมีปัญหานอกเวลาทำการ ติดต่อได้อย่างไร?+

ลูกค้าที่มีสัญญา MA หรือ Managed Services จะมีช่องทางติดต่อทีมซัพพอร์ตโดยเฉพาะ สามารถแจ้งปัญหาได้ตลอด 24 ชั่วโมง
